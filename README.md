# Getting Started

## Dependencies

* Java 26.0.1
* JavaFX 26.0.1
* JNativeHook 2.2.2

## Installing

* Download and install [Java 26.0.1](https://www.oracle.com/java/technologies/downloads/)
* Download JavaFX from [OpenJFX](https://openjfx.io/), [Orcale](https://www.oracle.com/java/technologies/downloads/javafx/), or [Gluon](https://gluonhq.com/products/javafx/)
* Downlaod JNativeHook from [Github](https://github.com/kwhat/jnativehook)
* Add the following to the build path:
      * jnativehook-2.2.2.jar
      * All jars under javafx-sdk-26.0.1/lib/

## Executing program
This project will be easiest to run in Eclipse.
Compile the project and run Main.java

### Clicker
* Changing the click timing
    * Edit the value in the text box labeled "Click Interval ms" to change the interval between clicks in milliseconds
* Clicking at current mouse location
    * Click the toggle key ( default of F1 ) to enable/disable the auto-clicker
* Clicking at a set positon
    * Right click anywhere on the screen or change the values in the text fields labeled "X pos" and "Y pos" to set a position
    * Check the "Use Position?" checkbox
    * Click the toggle key ( default of F1 ) to enable/disable the auto-clicker
* Clicking a set number of times
    * Change the value in the text field labeled "Click Count" to the desired number of clicks
    * Check the "Use Click Count?" checkbox
    * Click the toggle key ( default of F1 ) to enable/disable the auto-clicker
* Changing the toggle key
    * Input a new key into the text field labeled "Toggle Key"
    * Press "Change Key"
  
### Wrinkler
* Creating the circle
    * Press the "Build Circle" button
    * Follow the directions in the text under the "Build Circle" button
* Start clicking
    * Click the toggle key ( default of F3 ) to enable/disable the circle clicker
    * NOTE: The clicker will not get all of the wrinklers right away. The wrinklers will get popped as they naturally move arround the cookie
* Changing the toggle key
    * Input a new key into the text field labeled "Toggle Key"
    * Press "Change Key"
### Rebirth
* Recording the clicking positions
    * Press the "Start Recording" button
    * Follow the directions in the text under the "Start Recording" button
* Changing the toggle key
    * Input a new key into the text field labeled "Toggle Key"
    * Press "Change Key"
